# TicketPlus 搶票機器人 (MaoMaoBot Extension)

這是一個專為 TicketPlus (遠大售票系統) 設計的自動搶票 Chrome 擴充功能。它能夠自動化登入、掃描票區、選擇票數並進入結帳流程，幫助使用者提高搶票成功率。

## ✨ 主要功能

*   **自動登入**：預先設定帳號密碼，自動填入並送出登入表單。
*   **多種掃描模式**：
    *   **指定票價**：只搶購特定價格的票區。
    *   **全區掃描 (Grab All)**：掃描所有可用票區，不限價格。
    *   **優先票價**：在掃描時優先選擇設定的價格，若無則選擇其他符合條件的票區。
*   **智能狀態監控**：
    *   **自動刷新**：當無票或頁面卡住時自動刷新。
    *   **開賣等待**：偵測「開賣時間」狀態，自動等待並刷新。
    *   **錯誤處理**：自動處理系統錯誤對話框。
*   **效能優化**：
    *   **圖片阻擋**：搶票期間自動阻擋圖片載入，加快頁面速度。
    *   **狀態機架構**：使用穩健的狀態機模式管理搶票流程，減少卡頓與錯誤。

## 🚀 安裝說明

1.  下載此專案程式碼到您的電腦。
2.  開啟 Chrome 瀏覽器，進入擴充功能管理頁面 (`chrome://extensions/`)。
3.  開啟右上角的 **「開發人員模式」 (Developer mode)**。
4.  點擊左上角的 **「載入未封裝項目」 (Load unpacked)**。
5.  選擇此專案的資料夾 (`MaoMaoBot_extension`)。
6.  安裝完成後，瀏覽器右上角會出現擴充功能圖示。

## 📖 使用教學

1.  點擊瀏覽器右上角的擴充功能圖示開啟控制面板。
2.  **設定參數**：
    *   **目標網址**：輸入活動的購票頁面網址 (例如 `https://ticketplus.com.tw/order/...`)。
    *   **帳號/密碼**：(選填) 若需要自動登入請填寫。
    *   **張數**：選擇欲購買的張數 (1-4 張)。
    *   **優先票價**：(選填) 設定最想買的票價。
    *   **目標票價**：輸入想搶的票價 (如 `2800, 3200`)，或開啟 **「全區掃描模式」**。
3.  點擊 **「執行搶票」** 按鈕。
4.  機器人將會自動開啟分頁並開始運作。
5.  若需停止，請點擊控制面板中的 **「停止程式」**。

## 📂 檔案結構

*   `manifest.json`: 擴充功能設定檔 (Manifest V3)。
*   `popup.html` / `popup.js`: 擴充功能控制面板介面與邏輯。
*   `content.js`: 注入到目標網頁的核心邏輯，包含狀態機與 DOM 操作。
*   `mao6.png`: 擴充功能圖示。
